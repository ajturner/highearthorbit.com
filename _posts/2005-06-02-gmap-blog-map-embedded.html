I've finally gotten some time to finish up embedding a GMap in my sidebar showing the locations of my posts. Not very interesting at the moment, but I'm about to embark on a trip, so wanted to get it setup for that.

It's using the Geo Plugin by Owen Winkler, lots of info from Rancid Bacon's GMaps Standalone, and the example of photo map using bbc backstage.

The following files are required:

  • Geo Plugin
  • GoogleMap RSS Feed - modified for your website
  • XML Stylesheet
  • Wordpress

I'm trying to clean it up to make the bubble smaller. It's rather overwhelming in such a small window. For this reason I removed displaying the description and any icons.

While I would like to roll this into the Geo Plugin, I'm not sure when that will get done, so feel free to try out the files above. And if you have any questions, don't hesitate to ask.
