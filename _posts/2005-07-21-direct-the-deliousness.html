Del.icio.us direc.tor is a super-slick Ajax interface for managing del.icio.us bookmarks.

I've been using the foxylicious Firefox plugin which syncs my del.icio.us bookmarks to my Firefox bookmarks. However, that's limited to my desktop machine and also is a *lot* of bookmarks.

So, while I pull a GTD and bookmark sites I find interesting so I can subsequently forget about them rather than keeping them in a tab, wanting to read them, and then fretting because I forgot about it.
